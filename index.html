Polecenia głosowe

Polecenia głosowe pozwalają kontrolować Poradnik głosem. Po włączeniu tej funkcji, Poradnik będzie nasłuchiwał poleceń do momentu ręcznego wyłączenia, resetowanie dowodów nie wyłącza go. Aplikacja użyje domyślnego mikrofonu, więc upewnij się, że ten jest ustawiony poprawnie. Rozpoznawanie głosu będzie działać również gdy okno przeglądarki nie jest aktywne oraz gdy używasz tego samego mikrofonu w grze.
Dowody:
  • "ilość dowodów <num>" poziom trudności <num>" -> Ustaw liczbę dowodów
  • "dowód <dowód>" -> Zaznacz dowód jako znaleziony
  • "wykreśl dowód <dowód>" -> Wykreśl dowód
  • "wyczyść dowód <dowód>" -> Wyczyść status dowodu
  • "małpia łapa <dowód>" -> Oznacza dowód jako wykreślony przez małpią łapę
Prędkość:
  • "prędkość <prędkość>" -> Oznacz prędkość ducha
  • "wyczyść prędkość <prędkość>" -> Wyczyść prędkość ducha
  • "standardowe przyśpieszenie w zasięgu wzroku" -> Zaznacz standardowe przyśpieszenie w zasięgu wzroku
  • "wykreśl standardowe przyśpieszenie w zasięgu wzroku" -> Wykreśl standardowe przyśpieszenie w zasięgu wzroku
  • "wyczyść przyśpieszenie w zasięgu wzroku" -> Wyczyść standardowe przyśpieszenie w zasięgu wzroku
Zdrowie psychiczne:
  • "psychika <psychika>" -> Zaznacz poziom zdrowia psychicznego przy polowaniu
  • "wyczyść psychikę <psychika>" -> Wyczyść poziom zdrowia psychicznego przy polowaniu
Duchy:
  • "wykreśl ducha <ducha>" -> Wykreśl wybranego ducha
  • "wyczyść ducha <ducha>" -> Wyczyść status wybranego ducha (poza ukrytymi kartami)
  • "wybierz ducha <ducha>" -> Zaznacz wybranego ducha
  • "usuń ducha <ducha>" -> Usuń kartę wybranego ducha
  • "informacje o duchu <ducha>" -> Wyświetl informacje o wybranym duchu
Odliczanie:
  • "odliczaj okadzenie" -> Rozpocznij odliczanie okadzenia
  • "zatrzymaj odliczanie okadzenia" -> Zatrzymaj odliczanie okadzenia
  • "odliczaj następne polowanie" -> Rozpocznij odliczanie do następnego polowania
  • "zatrzymaj odliczanie następnego polowania" -> Zatrzymaj odliczanie do następnego polowania
Dodatkowe:
  • "pokaż filtry" lub "pokaż narzędzia" -> Przełącz pomiędzy filtrami i narzędziami
  • "modyfikator prędkości <procent>" -> Ustaw modyfikator prędkości ducha na 50%, 75%, 100%, 125% lub 150%
  • "pokaż mapę [<nazwa mapy>]" -> Otwórz zakładkę map i wyświetl wybraną
  • "zamknij mapę" lub "ukryj mapę" -> Zamknij zakładkę map
  • "zresetuj dziennik" or "zresetuj poradnik" -> Resetuj Poradnik
  • "wyłącz rozpoznawanie głosu" -> Wyłącz rozpoznawanie głosu

Prywatność

Polecenia głosowe są kierowane przez wbudowany interfejs programistyczny. Pomimo, że ta funkcja wymaga dostępu do mikrofonu, to Twój głos nie jest nagrywany ani przechowywany.
